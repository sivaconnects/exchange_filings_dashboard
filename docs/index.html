About This Project
100% Free. No subscription. No login required.
This open-source dashboard uses a real headless browser (Playwright) to physically visit NSE and BSE websites every 5 minutes — just like a human user — extracting every filing as it appears. Data is stored as JSON and served via GitHub Pages at zero cost.
01
Real Browser Scraping
Uses Playwright Chromium to open NSE and BSE website pages. JavaScript runs, cookies are set, exactly like a human. Data is extracted from what the browser receives.
02
XBRL File Parsing
NSE attaches XBRL/XML files to each announcement. We fetch and parse them to extract the real PDF attachment URL, full description, and filing metadata.
03
All BSE Categories
AGM/EGM, Board Meetings, Company Updates, Corporate Actions, Insider/SAST, New Listings, Results, Integrated Filings, Others — all 9 BSE categories covered.
04
AI Insight Engine
When ANTHROPIC_API_KEY is set in GitHub Secrets, Claude analyses all filings across 24 investment categories with Buffett, Lynch and Bogle frameworks.
05
5-Minute Live Updates
GitHub Actions runs every 5 minutes during market hours (9 AM–6 PM IST, Mon–Fri). Deduplication ensures only NEW filings are processed each run.
06
Setup: 3 Steps
1. Fork on GitHub. 2. Enable Pages (gh-pages branch). 3. Enable Actions. Live at username.github.io/exchange_filings_dashboard
