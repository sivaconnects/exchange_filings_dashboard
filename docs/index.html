ℹ️ About This Dashboard
100% Free. No subscription. No API keys needed. Open-source dashboard scraping NSE + BSE every 5 minutes via GitHub Actions. Data stored as JSON/CSV in the repo. Served via GitHub Pages. Cost: ₹0.
01
5-Min Live Scraping
GitHub Actions runs every 5 minutes during market hours. Only new filings (deduped by ID) are processed each run — no duplicates.
02
XBRL Parsing for NSE
Each NSE announcement comes with an XBRL/XML file. We parse it to extract the real PDF URL, full description, and submission timestamp.
03
All 9 BSE Categories
AGM/EGM, Board Meetings, Company Updates, Corporate Actions, Insider Trading/SAST, New Listings, Results, Integrated Filings, Others.
04
AI Insight Engine
Set ANTHROPIC_API_KEY in GitHub Secrets to enable Claude AI analysis across 24 investment categories with Buffett, Lynch & Bogle frameworks.
05
Multibagger Philosophy
Promoter buying, SME strong results, expansion board agendas, FII entry — these signals appear in filings before price moves.
06
3-Step Setup
1. Fork this repo. 2. Enable GitHub Pages (gh-pages branch). 3. Enable Actions. Your live dashboard is up instantly for free.
