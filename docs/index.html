How This Dashboard Works

100% Free. No subscription. No API keys. This is an open-source project that scrapes NSE and BSE exchange filings daily using GitHub Actions and publishes the results here via GitHub Pages. All data is stored as JSON/CSV files in this repository — free for everyone to use, fork, and improve.
01

Daily Web Scraping

A Python script runs every night at ~11:30 PM IST using GitHub Actions. It scrapes NSE and BSE for all corporate filings of that day — announcements, board meetings, results, actions, and shareholding patterns.

02

Data Stored as Files

All scraped data is saved as JSON and CSV files directly in the GitHub repository under data/. No database needed. You can download any day's data directly from GitHub.

03

Opportunity Scoring

Every filing is automatically scored for investment signals — promoter buying, bonus issues, buybacks, expansion plans, turnaround indicators, FII/DII entry patterns, and more.

04

Published Free

GitHub Pages serves this dashboard for free. GitHub Actions runs the scraper for free (2,000 minutes/month). The whole stack costs ₹0.

05

Multibagger Philosophy

The best opportunities are found before institutional investors enter. Promoter stake increases, strong SME results, unusual board meeting agendas — these are early signals the big money hasn't moved on yet.

06

Open Source Forever

Fork it, improve it, add your own signals. The goal is to democratize the research process that was previously available only to large investors with expensive Bloomberg terminals.
