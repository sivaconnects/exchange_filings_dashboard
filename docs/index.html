ℹ️ About This Project
100% Free. No subscription. No API keys needed to use. This open-source dashboard scrapes NSE and BSE exchange filings every 5 minutes using GitHub Actions — free on GitHub's public repo tier. Data is stored as JSON/CSV files directly in the repository. The dashboard is served via GitHub Pages at zero cost.
01
5-Minute Live Scraping
GitHub Actions runs every 5 minutes during market hours, checking for new NSE and BSE filings. Only new filings (deduplicated by ID) are processed each run.
02
XBRL Parsing for NSE
NSE filings come with XBRL files. We parse them to extract the real PDF attachment URL, full description, company details, and submission timestamp — giving you complete data.
03
All BSE Categories
BSE has its own 9 categories: AGM/EGM, Board Meetings, Company Updates, Corporate Actions, Insider Trading/SAST, New Listings, Results, Integrated Filings, Others. All are covered.
04
AI Insight Engine
When ANTHROPIC_API_KEY is set in GitHub Secrets, Claude analyzes all filings across 24 investment categories including Buffett, Lynch, and Bogle frameworks.
05
Multibagger Philosophy
The best opportunities are found before the crowd. Promoter buying, SME strong results, expansion board agendas, FII entry — these signals appear in filings days before price moves.
06
Setup in 3 Steps
1. Fork this repo on GitHub. 2. Enable GitHub Pages (gh-pages branch). 3. Enable Actions. Your live dashboard is instantly up at YOUR_USERNAME.github.io/exchange_filings_dashboard.
